# Inspector Pilot Checklist

## Pre-Pilot (Weeks 11-12)
- [ ] Inspector trained on mobile app (2-hour session)
- [ ] 5 dry-run inspections completed (practice mode)
- [ ] Inspector confirms app fits workflow ("Day in the Life" sign-off)
- [ ] Android device configured and tested

## During Pilot (Weeks 13-16)
- [ ] Daily check-ins with inspector
- [ ] Bug reports triaged within 24 hours
- [ ] Weekly feedback sessions

## Success Criteria
- **NPS:** ≥+20 (would recommend to colleagues)
- **Time savings:** ≥30% vs. paper-based workflow
- **Resubmission drop:** ≥20% reduction in applications requiring rework
- **Agreement rate:** ≥90% inspector agrees with auto-decisions

## Exit Conditions
- Either party can terminate pilot with 7 days notice
- If critical bug blocks workflow, pause pilot until fixed
