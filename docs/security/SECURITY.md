# Security Policy

## Reporting a Vulnerability
Email: security@permia.is

Do not open public issues for security vulnerabilities.

## Response Time
- **Critical:** 24 hours
- **High:** 72 hours
- **Medium:** 1 week
- **Low:** 2 weeks

## Scope
- Authentication bypass
- SQL injection
- XSS
- Evidence tampering
- Unauthorized data access

## Out of Scope
- DoS attacks
- Social engineering
- Physical access attacks
